Labyrinth - v1.2.0

This is a simple game based on the SFML libraries where you have to catch up with the enemy in order to kill him and collect coins.

Controls:

W, A, S, D, Arrows - Moving.

Tab - Show task info.

Space - Shooting. Shoot direction = player direction.

R - Restart game.

ESC - Exit.
